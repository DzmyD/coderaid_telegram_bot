# CodeRaid Telegram Bot
Телеграм бот, отправляющий коды из 4-х чисел каждому участнику кодрейда в игре __Rust__. Список всех вариаций 4-х значных кодов был взят [здесь](https://rusttips.com/top-10000-rust-door-lock-codes/).

## Инструкция по использованию:
1. Запустить бота на своем аккаунте.
2. Вводим команду `/gen x y`, где:
   * `/gen` отправка каждым сообщением по одному из вариантов 4-х значного числа;
   * `x` количество участников кодрейда;
   * `y` номер фрагмента списка чисел (числа будут генерироваться в порядке возрастания). Нужно заранее оговорить с остальными участниками какой номер будет присвоен каждому (начиная с 0, заканчивания `x - 1`);
3. При неудачной попытке удаляем сообщение с этим кодом и переходим на следующее.

## Запуск бота

1. Скачайте файлы [bot.py](https://github.com/DzmyD/coderaid_telegram_bot/blob/main/bot.py) и [list.txt](https://github.com/DzmyD/coderaid_telegram_bot/blob/main/list.txt) в одну папку.

2. Создайте __Telegram__-бота через официального [этого бота](https://telegram.me/BotFather). Если вы используете __@BotFather__ первый раз, вам необходимо его запустить нажатием кнопки "старт" внизу чата, либо ввести `/start`.

3. Используйте команду `/newbot`, далее следуйте указаниям бота.

<div align="center"><img width="500" height="532" alt="Untitled" src="https://github.com/user-attachments/assets/48214327-2886-48af-b425-bdd586632ef2" /></div>

4. В сообщении, которое начинается с фразы _Done! Congratulations on your new bot_, будет строка с __API__-ключом (в примере выше API-ключ замазан), вам необходимо скопировать его. __НИКОМУ НЕ СООБЩАЙТЕ ЭТОТ КЛЮЧ!!!__

5. Скопированный ключ необходимо вставить в новый файл формата `.txt`. Учтите, что в этом файле:
   * Кроме ключа никаких символов не должны быть (даже переносов строк);
   * Полное название файла `token.txt`;

6. Поместите этот файл в ту же папку, куда вы скачали файлы из первого пункта.

7. Предполагается, что у вас установлен __Python__-интерпретатор. Скачать установщик Python-интерпретатора можно [здесь](https://www.python.org/downloads/). Если возникли вопросы, вы можете прочитать эту [статью](https://practicum.yandex.ru/blog/kak-ustanovit-python-na-kompyuter/).

8. Запустите терминал (В __Windows__ нужно выполнить комбинацию клавиш `Win` + `R`, в появившимся поле ввести `cmd`).

9. В теминале выполните команду `pip install telebot`.

10. Для запуска скрипта бота необходимо запустить исполнение файла `bot.py`.
